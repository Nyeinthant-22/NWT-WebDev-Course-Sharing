What does a navigation bar do?
A navigation bar (nav bar) helps users move between different sections or pages of a website. It provides quick access to important links, making the site easier to use and improving the overall user experience.
How to create a navigation bar?
  1. Use the <nav> tag in your HTML.
  2. Place your links inside a <ul> (unordered list) with <li> (list items).
  3. Wrap each link with an <a> tag.
  4. Style the navigation bar with CSS for background color, spacing, and hover effects.
